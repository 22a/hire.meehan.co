Who am I?

I'm Peter Meehan, a 20 year old web, mobile, and full stack developer, halfway through my third year studying Computer Science at Trinity College Dublin. I'm also Dublin University Computer Science Society's Public Relations Officer. I have a penchant for mobile oriented design, scalable web applications, and vim.

I just so happen to be on the lookout for a summer internship this year on an exciting software development team like yours!

What have I done?

Last summer, a friend of mine and I were chosen by the school and hired to design and build the University's new digital student ID mobile applications. This involved conceptualising, building, testing, and publishing the Android and iOS applications from start to finish. Here we used a combination of Phonegap to build the mobile apps and Rails to build an API to interface between the students' phones and the college's student database.

At the start of the academic year it became apparent that DUCSS needed a set of new society sign up applications as the old version was built upon a since depricated Dropbox API. Here I used the Ionic Framework to build Android and iOS applications to securely send registration data off to a database behind an API.

As part of my involvement with DUCSS I have deployed and modified a build of Domjudge in order to allow our teams participating in programming competitions to compete in practice sessions on the same infrastructure that they will encounter at the competitions themselves. This involved fiddling with some unsightly PHP and cleaning up a few styles. Aside from the link below, more information can be found in the repo.

Many other open source tinkerings can be found over on my github!
